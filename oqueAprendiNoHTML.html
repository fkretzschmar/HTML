Na segunda etapa vamos stalar algumas expansões no VSCode abaixo:

Menu

!DOCTYPE

Se dentro do arquivo você aperta ! e aperta enter ele vai trazer o codigo base de
html5 veja exemplo abaixo.

Conforme imagem a cima o html é o corpo do site, dentro do html você pode colocar lang
aonde você informa a linguagem que sera utilizada no site então ficaria assim (<html lang="pt-br">;).

O <head> é aonde fica todas as configurações da página, achamos o charset lá que é um atributo que serve
para dizer pro nosso navegador qual vai ser a codificação de caracteres, cada idioma tem seus caracteres
mais para o navegador reconhecer os caracteres brasileiro utilizamos o UTF-8.

Dentro do <head> encontramos também o title aonde informamos o nome da página, que ficara escrito
la em cima na aba.

Por fim temos a tag <body> dentro do mesmo fica todo o conteudo do site que podemos interagir e visualizar.

HTML Entities

Entities serve para fazer simbolos dentro do HTML que podera ser visualizado no site.

Exemplo:1:<|2:>|3:∑|4:Δ

O código para fazer os exemplos acima inicia com & e termina com ; desconsidere o sinal de + no código.

Código:1:&+lt+;|2:&+gt+;|3:&+sum+;|4:&+Delta+;

Para aprender mais entities entre em w3schools

Emojis

Para você criar um emoji e seu texto basta você tem o codigo dele, você encontra varios deles na w3schools.

Se no código tiver o U desconsidere o mesmo, copie os numeros e coloque apos o &# não podemos esquecer do ; se não ele não reconhece.

Então ele vai ficar assim (&#+128516+;)desconsidere o sinal de +.

Favicon

Você já reparou ali em cima na frente do titulo da pagina a maioria dos sites tem um desenho
pequeno ele é chamado de favicon

Para você fazer isso é necessário baixar um favicon ou você pode criar o seu próprio favicon
no site favicon.io.

Após baixar e exportar o favicon insira o mesmo na pasta que esta seu arquivo html.

Em seguida você vai la em cima no <head>, em cima do title você escreve link e vai puxar
(link.favicon) então o código vai ficar conforme exemplo abaixo.

Código:

<link rel="shortcut icon" href="Aqui você aperta Ctrl+Espaço e escolha o favicon" type="image/x-icon">

Tag <p> e <br>

A tag <p> é aonde colocamos o texto que iremos apresentar, a tag <br>
serve para quebrar o texto para linha de baixo.

Segue dois exemplos abaixo o ex1 não vai ter a tag <br> o segundo ex2 vai ter.

Exemplo:

1:A Segunda Guerra Mundial foi um conflito militar global que durou de 1939 a 1945, envolvendo a maioria das nações do mundo — incluindo todas as grandes potências — organizadas em duas alianças militares opostas: os Aliados e o Eixo.

2:A Segunda Guerra Mundial foi um conflito militar global que durou de 1939 a 1945,
envolvendo a maioria das nações do mundo — incluindo todas as grandes potências —
organizadas em duas alianças militares opostas: os Aliados e o Eixo.

Tag <img>

Para colocarmos uma imagem em nosso site basta usar a tag <img> conforme código abaixo.

Código:

<img src="Aqui dentro vai o link da imagem ou a imagem no formato png ou jpeg no seu computador"

<img src="" height="aqui você marca a altura da imagem" width="nesse campo a largura"

<img src="" alt="Se a imagem der um erro você informa nesse campo sobre oque se trata a imagem"

img src="" title="Quando passar o mouse por cima da imagem vai aparecer oque for escrito aqui"

Código completo:<img src="https://cdn.pixabay.com/photo/2013/07/04/11/04/google-images-143148_960_720.png" alt="Imagem Google" title="Google" height="300" width="300">

Imagem Google:

Caso você queira colocar uma imagem que já tem no seu computador a mesma tem que estar na pasta do seu arquivo html.

Na organização do seu projedo você pode estar criando uma nova pasta para deixar as imagens.

Então o código ficaria assim:<img src="imagem/foto1.png">

É importante você sabe que ao cria um site você tem que verificar se as imagens utilizadas
é de domínio público pois você pode acabar se esbarrando nos direito autorais.

    <a href="#parte1">nome do link(Parte1)</a>

    Tag <ol> e <ul>

    A tag ol e ul serve para listar uma quantidade de itens, dentro da tag ol ou ul cada item deve ser separado dentro da tag por outra tag a <li> segue exemplo abaixo.

    Código:

    <ol>

    <li>Ana</li>

    <li>Felipe</li>

    <li>Carlos</li>

    </ol>

    Exemplo ol:

    1. Ana
    2. Felipe
    3. Carlos

    Exemplo ul:

    • Ana
    • Felipe
    • Carlos

    Tag <dl>

    A tag <dl> é usada para criar uma lista de definições ela lembra a tag ul e ol,
    dentro dela usamos a tag <dt> e <dd> segue exemplo abaixo.

    Código:

    <dl>

    <dt>HTML</dt>

    <dd>HTML é uma linguagem de marcação para criar conteúdo no site.</dd>

    <dt>CSS</dt>

    <dd>CSS é uma linguagem de marcação para criar design no site.</dd>

    </dl>

    Exemplo:

    HTML
    HTML é uma linguagem de marcação para criar conteúdo no site.
    CSS
